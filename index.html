About this Website

This is an experimental website created for the sole purpose of practicing my html and css skills. This website was built using the Twitter Bootstrap library. This website was designed to create an aesthetic display of my favorite car brands. This website is responsive as it will automatically resize the componenets based on the size of the screen. Currently, this website is not functional as the links on the page do not work. However, I hope to learn the Django framework in order to give my websites functionality in the future. I hope that this website can serve as a benchmark to see potential future progress.
